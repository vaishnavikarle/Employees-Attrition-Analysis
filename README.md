# Employees-Attrition-Analysis-Project
This project analyzes employee attrition to understand the key factors influencing employees leaving the organization. The analysis explores demographics, job-related factors, satisfaction levels and organizational measures that contribute to attrition. Findings are based on EDA, Statistical Analysis and Visualization.
